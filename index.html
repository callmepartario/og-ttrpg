Cypher System

The Cypher System is a setting-agnostic game publshed by Monte Cook Games.

This high contrast, form-fillable PDF character sheet allows for additional customization for your character or campaign. It is available in several languages. A full unofficial player's guide and quick-reference sheet are also available.

See also: Old Gus' Cypher System Reference Document (OG-CSRD)

Daggerheart

Daggerheart is a collaborative heroic fantasy game published by Darrington Press.

This high contrast, form-fillable PDF character sheet provides plenty of room to record all your character options through all ten levels of play, even if you don't own the official playset's card decks.

If you choose the Beastbound Ranger class, choose the version with the Ranger Companion. Otherwise, use the Extra Notes version.

See also: Old Gus' Daggerheart System Reference Document (OG-DHSRD)

Vagabond

Vagabond is a pulp fantasy game published by Land of the Blind.

This form-fillable PDF character sheet provides plenty of room to expand your spellbook, detail out class features, or take notes.

The assisted version populates many fields for you. If your PDF reader isn't compatible with these features, use the manual version instead. A blank version with no form fields is also provided.

Wolves Upon the Coast

Wolves Upon the Coast is an old school game published by Luke Gearing.

This high contrast, form-fillable PDF character sheet includes everything you need to create a character and start playing in minutes. A PDF player's guide is also avalable.

See also: Wolves Upon the Coast: The Unofficial Player's Guide (WUTC-UPG)
